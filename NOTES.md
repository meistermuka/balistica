## Initial Velocity:
```
v0 = v0_xi + v0_yj

Components:
v0x = v0 * cos(theta)
v0y = v0 * sin(theta)
```

## Kinematic quantities of projectile motion
### Acceleration
```
ax = 0
ay = -g
g = 9.81 m/s^2
```
### Velocity
```
vx = v0 * cos(theta)
vy = v0 * sin(theta) - g*t

Magnitude of velocity:
v = sqrt(vx^2 + vy^2)
```
